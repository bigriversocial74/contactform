# Stage 5C Storefront Management, Product Placement, and Public Preview

Stage 5C connects the Stage 4C public storefront foundation to the shared merchant workspace.

## Included

- Merchant storefront identity, slug, headline, description, contact, logo, cover, and theme settings
- Immutable draft, published, and retired storefront revisions
- Product placement with visible state, featured state, and display order
- Private merchant preview that prefers the current draft
- Public storefront rendering that uses only the published revision
- Published-product eligibility checks
- Storefront onboarding completion and payment-readiness unlock
- Shared Stage 5A shell and responsive design

## Boundaries

Feed visibility remains separate from storefront placement. Checkout and payment processing are not introduced. Public product pages continue using immutable product versions.
