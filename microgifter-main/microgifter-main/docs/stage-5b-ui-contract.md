# Stage 5B Merchant UI Contract

All Stage 5B product and media pages reuse the Stage 5A merchant shell, universal header, account menu, sticky sidebar, app panels, responsive drawer, buttons, badges, filters, empty states, and saved-state behavior.

The product list is the merchant operating index. The shared builder remains the edit surface. The product detail sheet is read-oriented and exposes current status, value, slug, builder type, immutable versions, and published media. The media library is an operational inventory, not a second uploader; uploads continue through the builder workflow so role assignments and draft state remain consistent.
