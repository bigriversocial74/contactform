# Stage 5H Acceptance Checklist

- Existing header notification and message dropdowns remain intact.
- Inbox integrates messages, notifications, and operational alerts in one workspace.
- Message threads remain gift- and PPPM-linked and user-scoped.
- Notification read state and unread badges stay synchronized.
- Users can manage per-type channel and digest preferences.
- Operational alerts support severity, acknowledgement, resolution, and dismissal.
- Message threads support pin, archive, and mute controls.
- Delivery jobs preserve channel, provider, retry, and delivery state without storing plaintext destinations.
- All writes require permission and CSRF protection.
- Stage 5H does not activate third-party email, SMS, or push providers.
- Desktop and mobile views follow the existing agent/account interface.
