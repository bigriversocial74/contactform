# Microgifter Build Downloads

Use numbered ZIP downloads in order.

1. `01_microgifter_stage1_foundation.zip` - initial Stage 1 identity foundation.
2. `02_microgifter_stage1_assets_integrated.zip` - Stage 1 foundation plus preserved uploaded builder/agent/index assets and upgraded auth UI.

Workflow:

1. We build or update files/code.
2. We save the code package to GitHub where possible.
3. A numbered ZIP is provided for download.
4. You upload/extract the ZIP into the repo or server.
5. The next ZIP increments the number.
