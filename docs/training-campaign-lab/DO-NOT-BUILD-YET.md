
## Not allowed right now

```text
PHP files

JavaScript files

Upload folders

Database migrations

```


Do not begin implementation unless the user clearly asks for code/build work, for example:

```text

Start implementation.
Create the PHP files.
```

Until then, continue improving docs only.
