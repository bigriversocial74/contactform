# Merchant Agent Creative Presets + Source Grounding

## Purpose
Adds merchant-facing creative preset chips and strengthens the merchant agent context so creative outputs are grounded in available account data and content sources.

## Presets
The Merchant Agent Chat composer now includes chips for:
- Social Post
- SMS
- Email
- Campaign Idea
- Reward Copy
- Local Event Promo

Each preset fills the composer and sets the matching output/scope controls.

## Grounding
Merchant Agent Chat now includes recent merchant feed post context alongside:
- merchant operating snapshot
- saved merchant memory
- memory source summaries and ready chunks
- merchant policy
- agent profile
- selected skills

The usage event metadata records whether feed posts were included and how many recent feed posts were available.

## SQL
No SQL required.
