# Store Canvas Messaging Summary

Phase 2 connects merchant Store Canvas messages to the existing platform messaging and notification layer.

Built:

- Merchant Canvas messages now create canonical message threads and messages.
- Customer notifications use the standard notification system.
- Store Canvas audit rows point back to the canonical message and thread.
- The Messages page displays source labels.
- Replies keep Store Canvas source metadata.
- Campaign rewards in the IN/OUT Box expose source metadata.
- Inbox, Sent, and Claimed views display source metadata on gift rows.
