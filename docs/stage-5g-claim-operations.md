# Stage 5G Claims, Merchant Verification, Claim Codes, Redemption, and Exception Operations

Stage 5G integrates merchant-defined location claim codes and the existing verification/redemption engine into the shared merchant workspace.

## Included

- Lookup by permanent gift ID or PPPM ID
- Merchant and location eligibility visibility
- Multiple reusable claim codes per location
- Claim-code creation, activation, deactivation, rotation, revocation, validity, and usage limits
- Verification followed by explicit redemption
- Failed-attempt and lockout visibility
- Claim history and audit timelines
- Exception operations for locked, expired, eligibility, duplicate, dispute, cancellation, and data issues
- Claim and PPPM linkage
- Responsive merchant-shell pages

## Identity boundary

The permanent gift/PPPM ID identifies the item. Merchant claim codes are reusable location credentials and never become item IDs. Plaintext claim codes are never stored or returned.
