# Stage 5B Supporting Notes

## Acceptance

Product lists and detail views are merchant-scoped; existing publish/archive behavior remains authoritative; immutable versions and historical asset references are preserved; the media library exposes processing and usage state; first publication advances onboarding.

## UI

Product, detail, and media views reuse the Stage 5A merchant shell and existing responsive account/agent/builder design language.

## Boundaries

Storefronts, orders, PPPM operations, distribution, claims, and payment processing remain in their planned later phases.
