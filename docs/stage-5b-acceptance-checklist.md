# Stage 5B Acceptance Checklist

- Merchant product list loads only the authenticated merchant's products.
- Search and draft/published/archived filters work.
- Product rows link to the shared builder and product detail sheet.
- Draft products can be published through the existing catalog lifecycle.
- Active products can be archived without deleting historical versions.
- Product detail shows immutable version history and current published media.
- Media library is owner-scoped and shows processing state and version usage.
- Publishing a first product completes the merchant onboarding `first_product` step and unlocks storefront setup.
- Published media and offer snapshots remain versioned; no in-place mutation is introduced.
- Product, detail, and media pages use the Stage 5A merchant shell on desktop and mobile.
